# SCiO Material Memory Android

Aprire questa cartella direttamente con Android Studio.

Funzioni principali:

- BLE scan e connessione GATT;
- ricerca UUID comando SCiO `3492`;
- notifiche automatiche nello stesso servizio;
- batteria, temperatura, scansione;
- assemblaggio tre sezioni;
- database SQLite locale;
- confronto sperimentale;
- esportazione JSON.

Guida completa: `../../docs/COMPILAZIONE_SU_MAC.md`.
